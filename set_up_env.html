Set Up Your Local Environment

To set up your local development environment, complete these steps:

  1. Install the necessary software as listed in Software Requirements.
  2. Clone this repository as described in Clone the nexus-6-ui Repository.
  3. Create a virtual environment as described in Create a Virtual Environment.
  4. Install and run the UI as described in Install and Run the UI on localhost.
  5. If you intend to deploy the UI to AWS, you must also install duo-sso as described in Install duo-sso.
