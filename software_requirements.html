Software Requirements

To contribute to this codebase, you must install the following software:

  • Python (> v3.8.8)
  • Node.js (v16.13.2)
  • Angular CLI (v13.3.10)
  • AWS CLI (v2.2.0) (Required for the deployment script.)
  • Git & Gitflow
  • SourceTree Git Client (Recommended but not required.)
